wie buchen Sie Freitauchen auf Fuerteventura

Apnoetauchen auf Fuerteventura zu buchen ist so einfach

Klicken Sie auf die Schaltfläche „Mit 10 % Rabatt buchen“ und Sie finden ein einfaches Formular, in dem Sie die Aktivität, den Tag und die Stunde der Tour sowie einige andere zusätzliche Optionen auswählen. Anschließend füllen Sie ein kurzes Formular aus, an dessen Ende Sie auf klicken Link zum Bezahlen mit Karte.

Wenn Sie beim Abschluss der Online-Buchung ein Problem haben, teilen Sie uns dies bitte per E-Mail oder Telefon mit.

MERKEN SIE

Eine Buchung für das Freitauchen auf Fuerteventura kann mindestens 6 Stunden vor Beginn der Aktivität erfolgen.

Wenn Sie in einer Privatwohnung übernachten, müssen Sie uns die genaue Adresse (und wenn möglich eine Stecknadel auf der Karte) mitteilen und am Tag der Freitauchtour bitte vor dem Gebäude auf den Transfer warten.

Denken Sie beim Schreiben Ihrer Kontakttelefonnummer daran, die Vorwahl Ihres Landes anzugeben.

Nach Abschluss der Buchung erhalten Sie eine automatische Bestätigung ohne Abholzeit. In Kürze senden wir Ihnen eine weitere E-Mail mit der genauen Abholzeit. Sollten Sie diese zweite E-Mail nicht am selben Tag von uns erhalten, kontaktieren Sie uns bitte.

Wenn Sie Last Minute buchen möchten, kontaktieren Sie uns bitte per Telefon oder WhatsApp.

Wenn Sie im Norden von Fuerteventura wohnen, können Sie unsere Freitauchschule mit dem Auto erreichen. Wir befinden uns in La Lajita (https://goo.gl/maps/Uoy5AVDfrfDNHsVn9, siehe Karte unten).

Stornierungsbedingungen

50 % Rückerstattung, wenn die Tour 24 Stunden oder früher vor Tourbeginn storniert wird.

0 % Rückerstattung, wenn die Tour weniger als 24 Stunden vor Tourbeginn storniert wird.
